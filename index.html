Introduction

This page is meant as a reference point for the MyUFV design system.

It is meant to be a living document, and will be updated as the design system evolves. We should treat this page and the style sheets in this repo as our source of truth for MyUFV styling.

Icons

We primarily use font-awesome icons throughout the application. As a fallback, we also use icons from Ellucian's design system.

If you choose to use font-awesome, you'll have to get your own license here.

Misc utilities

Components

Components are pre-built user interfaces that can be edited and dropped into our MyUFV components as needed. To keep things consistent, we've documented a few of them here.
